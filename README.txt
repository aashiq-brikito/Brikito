BRIKITO WEBSITE — HOSTING PACKAGE
=================================

Contents
  index.html          main site (landing page)
  how-it-works.html   interactive scroll demo page
  bkt-hiw.css         demo module styles
  bkt-hiw.js          demo module script
  images/             CCTV section images (4 files)

How to host
  Upload the WHOLE folder to any static host, keeping the
  structure exactly as is. Examples:

  - Netlify / Vercel: drag and drop this folder, done.
  - GitHub Pages: push these files to a repo, enable Pages.
  - Any cPanel / shared hosting: upload into public_html.

  No build step, no server code, no database needed.
  Fonts load from Google Fonts (needs internet).

  index.html is the entry point: yourdomain.com will open it
  automatically on every static host.
